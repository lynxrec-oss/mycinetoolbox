# YouTube SEO Best Practices (TubeBuddy-Based Rules)

When optimizing YouTube video metadata (titles, descriptions, tags, chapters) and visual thumbnails, adhere to the following checklist rules:

## 1. Keyword & Outline Integration
- **Target Keyword:** Define exactly one target keyword or key phrase per video.
- **Spoken Keyword:** Check if the target keyword is spoken naturally within the first 60 seconds and throughout the script/transcript. YouTube's automated transcription service indexes spoken audio to verify relevance.

## 2. Title Optimization
- **Keyword Placement:** Naturally include the target keyword phrase early in the title, ideally within the first 60 characters.
- **Phasing & Match:** If the exact phrasing is awkward, include at least 80% of the target keyword phrase.
- **Length:** Keep titles between 53 and 70 characters (preferred for maximum CTR without search layout truncation). The hard title limit is 100 characters.

## 3. Description Tuning
- **The Hook Snippet:** Include the target keyword naturally within the first 200 characters of the description (which displays as the search result snippet).
- **First Two Sentences:** The first two sentences must clearly describe the video and include the keyword organically.
- **Body & Length:** The description should be about 200–300 words long to feed the YouTube search crawlers plenty of semantic context.
- **Variations:** Use natural synonyms, variations, and partial matches. Avoid mechanical repetition/keyword stuffing.

## 4. Tag Configuration
- **Primary Tag:** The exact target keyword phrase must be the very first tag.
- **Variations:** Add 3–5 closely related tag variations and long-tail phrases.
- **Relevance:** Add only relevant broad, specific, and long-tail tags. Favor phrase-based tags over generic single-word tags.

## 5. Visual Thumbnails
- **Resolution:** Thumbnails must be custom, high-resolution (at least 1280 × 720 px, under 2MB).
- **Legibility:** Text, subjects, and graphics must be highly legible when scaled down to mobile screen previews.
- **Honesty:** The thumbnail must accurately depict the video content and avoid being misleading (clickbait).

## 6. Timestamped Chapters
- **Longer Videos:** Add timestamped chapters for videos longer than 5 minutes to trigger Google Search's "Key Moments" listings.
- **Chapter Keyword:** Include the target keyword naturally in at least one chapter title.

## 7. Performance & Optimization Loop
- **Score Verification:** Run a final score check for title, description, tags, thumbnail, chapters, and keyword alignment before publishing.
- **Audit Analytics:** Check the Click-Through Rate (CTR) at 24 hours and Average View Duration (AVD) at 48 hours post-publish.
- **Refinement:** If CTR is below the channel average, test a new title or thumbnail layout.
